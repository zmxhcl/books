### 一，验证npm

在dos窗口中输入``npm -v``,如下图显示，即为成功。

![2.2](..\asstes\2.2.png)



### 二，cnpm安装

在dos窗口中输入`` npm install -g cnpm --registry=https://registry.npm.taobao.org``,就可以自动安装。

#### 注意：

* 设备网络波动会导致安装过程被中断，重新运行上述指令即可。
* 安装过程因网络原因，长短不定。



### 三，cnpm验证

在dso窗口中输入``cnpm -v``,如果出现下图效果，即为安装成功

![2.3](..\asstes\2.3.png)
