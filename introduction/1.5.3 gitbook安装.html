1.5.3 gitbook安装

一，gitbook安装

在dos窗口中输入cnpm install gitbook-cli -g

1.1 指令解析

cnpm 安装工具

install 安装的意思

gitbook-cli 是我们即将安装的软件名字

-g g是global的缩写，是全局的意思

1.2 验证安装

没有报错就是成功了，哈哈哈哈哈~~~~
